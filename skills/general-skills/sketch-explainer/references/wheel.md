# Format: Wheel / Radial Diagram

## When to use
Use this format for topics where **multiple equal or related elements all connect to or revolve around a central concept** — and where no single element is "first" or "last". Great for: ecosystems, pillars of a framework, the components of a system that all serve a core goal, skills in a discipline, values of an organization, aspects of a concept.

**Keywords that suggest wheel:** "components of", "pillars of", "ecosystem", "aspects of", "what makes up", "elements of", "types of [central thing]", "dimensions of", when there's a clear center concept with 4–8 surrounding ideas

## Layout description

A central hub with spokes radiating outward to labeled segments on a white background, hand-drawn in whiteboard sketch style.

**Core anatomy:**
- **Center circle**: bold, slightly imperfect hand-drawn circle containing the core concept name (1–3 words), filled with the most prominent pastel color
- **Spokes**: straight-ish hand-drawn lines radiating from the center, one per concept (typically 4–8)
- **Outer segments or bubbles**: at the end of each spoke, either:
  - A rounded rectangle "petal" (for a petal/flower layout)
  - A curved arc segment (for a true pie-wedge wheel)
  - A circle or oval bubble (for a hub-and-spoke with floating nodes)
- **Each segment contains**: a bold label, a small sketch icon, and optionally a 1-line caption beneath it
- **Colors**: each segment/petal gets its own pastel color, cycling through the palette

**Optional inner ring**: a thin ring between the hub and the outer segments, subdivided into small arc labels that categorize the spokes into 2–3 groups.

**Structure:**
- Title: large sketchy handwritten lettering at the top (or bottom, to not crowd the wheel)
- Wheel centered on the page
- Even spacing between spokes (try to keep angles roughly equal)
- Labels readable (not rotated awkwardly — if using arc segments, place labels horizontally inside them)

**Visual feel:** Organic and balanced. The wheel communicates "everything connects to this center" intuitively. Hand-drawn wobble gives warmth. Not a pie chart — a diagram.

## Prompt structure

```
[Title] — hand-drawn whiteboard wheel diagram titled "[Topic]" in large sketchy lettering at the top.

White background. Radial wheel layout centered on the page. Pastel color scheme. Hand-drawn sketch aesthetic.

Center circle: slightly imperfect hand-drawn circle filled with salmon `#F4A896`, labeled "[core concept]" in bold hand-lettered text.

[N] spokes radiate outward at equal angles. At the end of each spoke, a rounded petal/bubble shape:

Spoke 1 (soft yellow `#F7E08A`): label "[concept 1]", icon [icon description], caption "[one-line description]"
Spoke 2 (sage green `#A8D5A2`): label "[concept 2]", icon [icon description], caption "[one-line description]"
Spoke 3 (slate blue `#A8C5DA`): label "[concept 3]", icon [icon description], caption "[one-line description]"
Spoke 4 (lavender `#C5B8E8`): label "[concept 4]", icon [icon description], caption "[one-line description]"
[... continue for each spoke]

Spokes are slightly wobbly hand-drawn lines. Petal shapes have slightly irregular rounded borders.

Sketch aesthetic: hand-lettered labels, friendly and approachable, educational whiteboard style. No gradients, no drop shadows, no corporate polish.
```

## Color assignment guidance

- Center hub: salmon `#F4A896` (most prominent, draws the eye)
- Outer segments: cycle through yellow → green → blue → lavender → salmon (if >5)
- Spoke lines: dark charcoal, hand-drawn weight, slightly irregular
