# Format: Linear Steps

## When to use
Use this format for topics that are **sequential, ordered processes where each step must happen before the next** — no branching, no hierarchy, just a clear numbered progression. Great for: tutorials, recipes, setup guides, lifecycle stages, timelines, historical progressions, how something is built or assembled.

**Keywords that suggest linear steps:** "how to", "step by step", "process", "stages", "phases", "timeline", "lifecycle", "in order"

## Layout description

A horizontal or vertical chain of numbered steps on a white background, hand-drawn in whiteboard sketch style.

**Two sub-variants — pick based on step count:**
- **Horizontal ribbon** (3–5 steps): Steps flow left to right, connected by arrows. Works well when steps have short names.
- **Vertical stack** (5–8 steps): Steps flow top to bottom as numbered blocks. Better when each step needs more explanation.

**Shape anatomy (per step):**
- A rounded rectangle or lozenge for each step, each in a distinct pastel color (cycle through the palette)
- A large bold number (1, 2, 3…) in the top-left corner of each block
- A short bold label in the center (3–6 words)
- A small italic caption below the label describing what happens in that step
- A small sketch icon (gear, arrow, magnifying glass, checkmark, etc.) to the right or top of the label
- Connecting arrows between steps: thick, slightly wobbly, pointing right (horizontal) or down (vertical)

**Structure:**
- Title: large sketchy handwritten lettering at the top
- Steps flow in reading order (left-to-right, or top-to-bottom)
- Optional: a thin baseline or "track" line running under all steps (like a timeline rail)
- Optional: milestone markers or phase labels grouping steps into phases

**Visual feel:** Clean, scannable, educational. The numbered steps make the order unmistakable. Hand-drawn but organized.

## Prompt structure

```
[Title] — hand-drawn whiteboard step diagram titled "[Topic]" in large sketchy lettering at the top.

White background. [Horizontal ribbon / Vertical stack] layout with [N] numbered steps connected by hand-drawn arrows. Pastel color scheme.

Step 1 ([color]): Large bold "1" in corner. Label: "[step name]". Caption: "[what happens]". Icon: [icon description].
Step 2 ([color]): Large bold "2" in corner. Label: "[step name]". Caption: "[what happens]". Icon: [icon description].
[... continue for each step]

Connecting arrows between each step are thick and slightly wobbly, pointing [right / down].

Sketch aesthetic: slightly imperfect lines, hand-lettered labels, educational whiteboard style. No gradients, no drop shadows, no corporate polish.
```

## Color assignment guidance

Cycle through the palette in order per step:
1. Salmon `#F4A896`
2. Soft yellow `#F7E08A`
3. Sage green `#A8D5A2`
4. Slate blue `#A8C5DA`
5. Lavender `#C5B8E8`

If there are more than 5 steps, restart the cycle. Arrow lines: dark charcoal, hand-drawn weight.
