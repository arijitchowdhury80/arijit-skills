# Blog Post Template

## Structure

### 1. Title (H1)
- Sentence case (capitalize only first word + proper nouns)
- No period at end (question marks OK)
- 50-70 characters for SEO
- Clear, specific, and value-driven

### 2. Meta Description
- 150-160 characters
- Include primary keyword naturally
- End with a value proposition or call to action

### 3. Introduction (2-3 paragraphs)
- Hook: Start with a compelling stat, question, or pain point
- Context: Frame the problem or opportunity
- Promise: Preview what the reader will learn

### 4. Body Sections (H2 headings)
- Each section addresses one main idea
- Use H3 subheadings for subsections
- Include relevant examples, data points, or case references
- 2-3 sentences per paragraph max
- Use bullet points for lists of 3+ items

### 5. Conclusion
- Summarize key takeaways
- Include a clear CTA (try Algolia, read related content, download resource)
- Don't introduce new concepts

### 6. CTA Block
- Clear, action-oriented button text
- Link to relevant next step (sign up, demo, related content)

## Style Rules for Blog Posts

- **Tone**: Educational, authoritative, engaging
- **Length**: 800-1,500 words (adjust based on depth)
- **Audience**: Specify in the intro (developers, business users, etc.)
- **Numbers**: Use specific data — "1.75 trillion searches" not "lots of searches"
- **Links**: Link relevant nouns, not articles. Don't link to competitors.
- **Images**: Reference Algolia digital assets where appropriate
- **Headings**: Sentence case, no periods
- **Oxford comma**: Always
- **Language**: American English

## Blog Post Types

| Type | Focus | Example Title |
|------|-------|---------------|
| Thought leadership | Industry trends and insights | How AI is reshaping product discovery |
| Technical tutorial | Step-by-step implementation | Building semantic search with Algolia |
| Product update | New feature announcement | Introducing Agentic Retrieval |
| Customer story | Success narrative | How [Customer] increased conversion by 40% |
| Best practices | Actionable advice | Five search optimization strategies for e-commerce |
