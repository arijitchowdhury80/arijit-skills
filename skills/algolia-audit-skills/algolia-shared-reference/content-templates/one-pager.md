# One-Pager / Datasheet Template

## Structure (Single Page — Front Only)

### Header
- Algolia logo (top-left)
- Document title (H1, Sora Light)
- Subtitle or tagline

### Hero Statement
- 1-2 sentence value proposition specific to the product/feature/use case
- Large, prominent text

### Key Benefits (3-4 items)
- Icon + Bold heading + 1-2 sentence description
- Focus on outcomes, not features
- Include a metric if available

### How It Works (Optional)
- 3-step visual flow or architecture diagram
- Brief label for each step

### Key Metrics / Social Proof
- 2-3 standout numbers in large display text
- Customer logos or analyst recognition

### CTA
- Clear next step: "Start your free trial," "Request a demo," "Talk to sales"
- Contact info or URL

## Design Rules
- **Single page only** — maximum information density without clutter
- **Use brand colors**: #003DFF for accents, #23263B for text
- **Sora font** throughout
- **White background** preferred (dark sections acceptable for emphasis)
- **Generous whitespace** — don't cram
- **One main visual** — product screenshot, diagram, or illustration
