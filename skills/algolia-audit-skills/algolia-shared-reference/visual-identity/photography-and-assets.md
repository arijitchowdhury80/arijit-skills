# Algolia Photography & Digital Assets

## Photography Style

Algolia uses photography to visually put an emphasis on our customers' industries. Photography should be used with an overlay of one of our brand colors (typically the blue-to-purple gradient).

### Photography Guidelines
- Focus on customer industries and real-world use cases
- Apply brand color overlay to maintain visual consistency
- Show real products, real interfaces, real people when possible
- Photography examples include: retail/e-commerce (Dunelm), sports/lifestyle (Gymshark), marketplace (Mercari)

### Photography Overlays
- Always apply a blue or gradient overlay to raw photography
- This creates visual consistency across all brand materials
- The overlay ensures the Algolia brand colors are present even in photographic content

## Digital Assets

### Purpose
These illustrations and UI examples were created for the website. They can be reused in materials such as presentations to help reinforce a point.

**Important: All images belong to Algolia.**

### Asset Categories

#### Light Background Assets
- UI component illustrations
- Dashboard mockups
- Search interface examples
- Feature illustrations
- Data visualization graphics
- Integration diagrams

#### Dark Background Assets
- Same types as light background, optimized for dark contexts
- Use for dark-themed slides, dark sections of websites

### Usage Rules
- Use digital assets to illustrate points in presentations and collateral
- Don't modify or alter the digital assets
- Don't extract elements from composite assets
- Maintain original aspect ratios
- Use the appropriate variant (light/dark) for your background
