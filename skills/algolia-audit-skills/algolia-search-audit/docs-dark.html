What is the Search Audit?

The Algolia Search Audit is an end-to-end intelligence and testing system for Algolia sales teams. Given a prospect's website URL, it automatically researches the company, tests their search experience in a real browser, identifies gaps Algolia can solve, and produces a complete package of sales deliverables — all sourced, fact-checked, and publication-ready.

It is designed to replace hours of manual research with a single command, while producing output that is more thorough, consistent, and credible than anything built by hand.

Primary audience: Algolia Account Executives, Sales Engineers, and BDRs preparing for discovery calls with e-commerce and digital commerce prospects.

What it covers

  • Company intelligence — traffic, tech stack, financials, hiring signals, investor quotes
  • Live browser testing — 20 search behavior tests with screenshots against the live site
  • Scoring — 10-area search quality scoring (0–10 scale) with gap identification
  • ROI modeling — 6-component business case with conversion, AOV, and bounce metrics
  • Deliverables — deck, landing page, AE brief, signal brief, sales playbook, book PDF
  • Quality validation — automated fact-check across 7 dimensions before delivery

How it works

1
Invoke the orchestrator
Trigger with a natural language message like "run a full algolia search audit on dsw.com". The orchestrator skill activates and creates a workspace.
2
Phase 1 — Research wave (4 parallel agent teams)
6 intelligence sub-skills run in parallel: company context, tech stack (BuiltWith), traffic (SimilarWeb), financials (Yahoo Finance), hiring signals, and investor intelligence from 10-K/earnings calls.
3
Phase 2 — Browser audit (Chrome automation)
Real Chrome executes 20 standardized search behavior tests — NLP queries, typo tolerance, zero-results states, autocomplete, faceting, recommendations, mobile — capturing screenshots at each step.
4
Phase 3–5 — Score, model, produce
10-area scoring, 6-component ROI business case, full McKinsey Pyramid deck (~33 slides), dual-view landing page, AE pre-call brief, and strategic signal brief.
5
Quality gate — Factcheck & Eval
Every deliverable passes through a 7-dimension factcheck that re-calls APIs, follows every citation link, verifies investor quotes at source, and issues a PROCEED / WARN / BLOCKED verdict.

Installation & Setup

Install the skill suite

# Clone the distribution repo
git clone https://github.com/arijitchowdhury80/algolia-claude-skills.git

# Run the installer
cd algolia-claude-skills
chmod +x install.sh && ./install.sh

The installer copies all 13 skills to ~/.claude/skills/ and detects missing MCP servers.

Environment variables

export ALGOLIA_AUDIT_DIR="/path/to/your/audit/workspace"
export CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1  # enables parallel Phase 1

Verify installation

# In Claude Code, type:
/algolia-search-audit --help

Five MCP servers are required. The installer detects and warns if any are missing.
